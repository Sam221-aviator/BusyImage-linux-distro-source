Making Simple Linux Distro from Scratch

Cloning the linux github repository.

Copy this command in your terminal.

git clone --depth 1 https://github.com/torvalds/linux.git

After copying the github repository, type this.

cd linux && make menuconfig

Exit the menuconfig and save the changes. Now type this.

make -j 8

Explaination of make -j 8

This command will let the make command spilt into 8 different jobs. This command will be based of your operating system's number or sometimes your can type 4 as 8. That will work the same too!

Create a folder outside of your linux directory or it can also be named as boot-files.
