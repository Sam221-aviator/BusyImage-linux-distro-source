Booting with QEMU

After copying both of bzImage and init.cpio file inside your storage folder then we can run it.

qemu-system-x86_64 boot

After it booted, write this inside the SYSLINUX bootloader.

/bzImage -initrd=/init.cpio

Wait for a while and it might boot into the init file as the prompt like #

It also has vi, bc and others packages.

If you want to install languages like Lua, go to nirlichtman's YouTube channel and there might be a video called Making the tinest possible linux distro
